=== AI SEO Meta Generator (Gemini) ===
Contributors: PixelNinjaByte
Tags: seo, ai, gemini, meta description, open graph
Version: 1.0

A WordPress plugin that uses Google Gemini AI to automatically generate SEO meta descriptions and Open Graph meta tags. Includes an admin panel with API key input and an AI SEO playground.

== Features ==
* Auto-generate meta description using Gemini
* Auto-save SEO description to post meta
* Admin settings page with API key input
* AI SEO playground for testing prompts
* Instructions included inside the plugin page

== Installation ==
Upload and activate.
